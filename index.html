Stopwatch/Timer Count up -OR- down... Why? I'uuno, just do it! How do you do it You may ask... Well, putting aside the hysterical assumption you do not know how to work a stopwatch or timer, let me explain. All you need to do is set the time (provided you want to count down or, for some odd reason, count up from a certain point in time.) Too lazy to set the time? No problem. Just hit the "Count" button and it will count the seconds you have wasted playing with this tool all the way from 0. :) Want to count down? Again, no problem! Just "Pause" the current count first and then count down from that very point! Running marathons to lose some extra fat? See how you improve in each lap by saving up to 5 different counts of the time! Whether up or down, it is recorded for your personal use. Close the window when you are bored of using it though which will probably be before you count to 5. Enjoy! :)
